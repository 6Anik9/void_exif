## 💀 Tactical Capabilities

* **Live Kernel Readout:** Immediate extraction of hidden EXIF headers (Device Model, Software, Timestamps) directly into the terminal buffer.
* **Precision Injection:** Seamlessly overwrite identity tags with custom signatures (e.g., `6ANIK9_VOID`) for branding or privacy.
* **GPS Logic:** Advanced coordinate mapping that converts standard Latitude/Longitude into encoded EXIF rational formats.
* **The VOiD Purge:** Deep-clean images by stripping all metadata headers for maximum digital anonymity.
* **Visual Synchronization:** High-fidelity UI featuring a real-time Matrix canvas and satisfying pulse-ring animations.

## 📂 System Architecture

The VOiD Engine is built with a zero-footprint philosophy, ensuring all data remains on your local machine:

* **`index.html`** — The tactical structural frame and UI components.
* **`style.css`** — Neon-glass visual customization and smooth animations.
* **`script.js`** — The high-power V4 engine for metadata processing and binary handling.

## 🛡️ Zero-Trace Security

Following the same security standards as **Career Canvas**, all operations are executed within the **client-side memory buffer**. Your images never leave your device; the injection happens purely in the local browser kernel. No tracking. No logs. No trace.
